Sudoku Solver

AI sudoku solver implemented using an ARC-3 Consistency Algorithm. When run, the user is prompted to choose either a preloaded board or they can input their own. There are 5 levels of preloaded boards. The program is able to solve (or determine a lack of solution) for all puzzles attempted in ~2-3 seconds. If you'd like to try it out for yourself, the source code is available here!
